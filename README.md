**Análisis de Readmisión en Diabetes**

Análisis end-to-end de un dataset hospitalario centrado en la **readmisión a 30 días**.  
Incluye limpieza de datos, análisis exploratorio, modelado predictivo y visualización en Power BI.

**Resultados clave**

- **Tasa de readmisión <30 días:** 11.2%
- **Polifarmacia:** 79.8%
- Los **ingresos previos** muestran mayor asociación con la readmisión que variables clínicas aisladas.

**Tecnologías**

- Python (Pandas, NumPy, Scikit-learn)
- Jupyter Notebook
- Power BI
- Git

Proyecto que integra **experiencia sanitaria y análisis de datos** aplicado a un problema clínico real.
